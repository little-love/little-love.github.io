h5-表单

表单
input标签新输入类型

  • 左面看得到效果的
    • 颜色 type=”color”
    • 日期 type=”date”
    • 滑块 type=”range”
  • 移动端弹出不同的键盘的
    • 数字 type=”number”
    • 手机 type=”tel”
    • 邮箱 type=”email”
    • 搜索框 type=”search”
      对比之前css表订单类型，这个会更加的完善，提高用户体验度。
      新表单事件
  • input
    • 输入的时候会一直触发
  • change
    • 值改变时候会触发
    • range滑块停止滑动时候
  • invalid
    • 验证失败时候会触发

当表单验证失败时候，会显示设置的这个内容。

新表单元素
output
跟span差不多，只是语义更强一些。

新表单属性
placeholder 占位符
autofocus 自动获取焦点
multiple
是给type=“file” 一次性可以选择多个文件使用的表单属性

表单验证
自带有验证规则的
tyep=”email” 具有自带验证是否有@ .等格式的，但是不能验证是否为空
非空验证
required 直接在标签里加上就可以检验这个input为空的时候，会提示。
自定义验证规则
pattern=”\d{4}” 通过这样格式来验证是否符合规范
